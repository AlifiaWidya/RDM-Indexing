# RDM-Indexing
### -> image-search-engine
    * Clone terlebih dahulu dengan menggunakan syntax : 
       `git clone https://github.com/kudeh/image-search-engine.git`
    2. `sudo apt-get update`
    3. `sudo apt-get upgrade`
    4. `sudo apt-get install python3-pip`
    5. `cd image-search-engine/` lalu `pip3 install -r requirements.txt`
    6. `cd app` lalu `python3 index.py --dataset static/images --index index.csv`
    7. cek hasil dengan menggunakan syntax `pico index.csv`
